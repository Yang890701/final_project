## 🖥️ 本機快速啟動（不需任何帳號）
詳見 GitHub 的 `README.md`。三個終端機分別啟動：
```
cd backend-api && pip install -r requirements.txt && uvicorn app.main:app --reload   # :8000
cd backend-node && npm install && node src/server.js                                  # :3000
cd frontend     && npm install && npm run dev                                         # :5173
```
未設 DATABASE_URL / GEMINI_API_KEY 時自動以種子資料 + 規則 fallback 運作，整套仍可 demo。

Vue 3 前端 ＋ Node.js(Express，模擬遊戲) ＋ Python FastAPI(偵測模型 + Gemini + 統計) ＋ PostgreSQL；
